  • 5-second fade-out effect on music
  • Professional audio overlay mixing
Supported Audio Formats:
MP3 WAV OGG FLAC M4A AAC WMA
API Usage

You can also use the API programmatically:

curl -X POST -F "speech=@speech.mp3" -F "music=@background.mp3" \
     http://localhost:5000/mix --output mixed_audio.mp3
Endpoint: POST /mix
Content-Type: multipart/form-data
Parameters:
  • speech - Speech audio file
  • music - Background music file
Response: Mixed audio file (MP3 format)
